# -Docker-APM
Apache + PHP + Mysql with Docker!

## Quickstart

1. repository clone

    ```bash
    $ git clone https://github.com/dantory/-Docker-APM.git
    ```
2. run script

    **Please note that the script must be run inside the `docker` directory**

    ```bash
    $ cd docker
    $ sh ./scripts/create.sh
    ```

## Stop & Destroy Docker

- run destroy script

    ```bash
    $ sh ./docker/scripts/destroy.sh
    ```

**Enjoy!**
